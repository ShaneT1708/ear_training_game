Relative Pitch Ear Training: Instructions

Use the sliders to select the number of notes to be played in sequence, and the rate of playback.

Choose from the 3 modes. Since there are fewer black keys, it is the easiest option.

Black Keys = Pentatonic, White Keys = Diatonic, All Keys = Chromatic

Press "Play!" to generate the melody. Press "Repeat" to hear it again. If you want a new melody to be generated press play again.

The first note is highlighted, you must enter all the following notes in the right order.

Press "Submit" to check if you are correct, "Undo" to undo your last note and "Reveal" to reveal the correct answer
